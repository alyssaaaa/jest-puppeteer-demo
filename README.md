# jest-puppeteer-demo

jest-puppeteer-demo is a demo for UI Testing with Jest and Puppeteer. 

### Installation

jest-puppeteer-demo requires [Node.js](https://nodejs.org/) v7.6.0+ to run.

Install the devDependencies and run the tests

```sh
$ yarn install
$ yarn test
```
